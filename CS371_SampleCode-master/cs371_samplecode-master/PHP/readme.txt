Sample code for several PHP-related topics:

helloWorld.php:
  Very simple first example.

bug.php:
  Check whether PHP environment will display error messages.  It is
  common to suppress error messages in a production environment; but,
  you definitely want to see them when learning PHP.

globals.php:
  Shows all the variables that are automatically set up and available
  for your use.  Also demonstrates how the key/value pairs in query
  strings are placed into variables.

formDemo.php:
  Demonstrates (1) How to set up a simple HTML form, and (2) How to use
  PHP to access the data.

myCustomRouter:
  Sample routing script.  This type of script is used by frameworks such
  as Laravel.

index.php:
  Sample default web page (used with URL points to missing file).
