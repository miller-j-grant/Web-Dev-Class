CS 371 Midterm Coding

PHP

Write a PHP script that will generate a mileage reimbursement invoice.
  • Load the data from invoice_data1.txt.
  • Use CSS to style the document neatly. Your document should be nicer than the example.
    • Make sure dollar signs appear in the appropriate places.
    • Add lines as shown in the sample below.
    • Make sure numbers are right-justified and text is either left justified or centered, as appropriate.
    • Make sure there is a reasonable amount of space between the columns (i.e., don't let the columns crowd together).
  • The per-mile reimbursement rate depends on the speed of the particular leg. Write and use a separate function that takes the speed as input and returns the appropriate rate. You can hard-code the values below, or you can load them from this file.
    Speed RangeRate
    75 or above:$0.15
    70 - 75$0.45
    65 - 70$0.55
    60 - 65$0.50
    50 - 60$0.40
    40 - 50$0.30
    below 40$0.15
Hint: Remember the PHP "explode" and "number_format" functions.
